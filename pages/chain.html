How to Bootstrap the MIMI Money Blockchain

This guide will help you install a recent copy of the blockchain. This should significantly speed up the task of getting the blockchain synced up so you can use your wallet.

Windows:

  1. Make sure MIMIMoneyd.exe, mimi-service.exe, and/or the GUI wallet are not running.

  2. Open File Explorer and type %APPDATA%\MIMI Money and hit enter.

Note: if the folder doesn't exist, just go to %APPDATA% instead and create a folder named MIMI Money.

  1. Delete the following if they exist:

  • blockindexes.bin

  • blocks.bin

  • "DB" folder

    Note: In case it is unable to delete the files due to them being used by some other program, follow these steps:

    • Open Task Manager with the shortcut Ctrl + Shift+ Escape.
    • Click on Processes.
    • Click on Image Name.
    • Scroll to the bottom.
    • Click on MIMIMoneyd.exe.
    • Click on End Process.
    • Click on mimi-service.exe.
    • Click on End Process.
    • Try to delete them again.

  1. Download the latest bootstrap snapshot of the blockchain.

  2. Move the two new downloaded files to the %APPDATA%\MIMI Money folder.

  3. Start MIMIMoneyd.exe or the GUI wallet like you normally do.

  4. See Expected Results section below.

Mac & Linux:

  1. Make sure MIMIMoneyd, mimi-service, and/or the GUI wallet are not running.

  2. Open Finder.

  3. Use the shortcut Command + Shift + G to bring up Go to Folder:

  1. Delete the following if they exist:

  • blockindexes.bin

  • blocks.bin

  • "DB" folder

  1. Download the latest snapshot of the blockchain.

  2. Move the two new downloaded files, blockindexes.bin and blocks.bin into the ~/.MIMI Money/ folder.

  3. Start MIMIMoneyd or the GUI like you normally do.

  4. See the Expected Results section below.

When you start MIMIMoneyd you should see this. Note that the blocksize (120369) is just an example & will be a different number at the time you perform the bootstrap:

2018-Feb-01 18:43:37.225492 INFO    Importing blocks from blockchain storage
2018-Feb-01 18:43:37.741587 INFO    Imported block with index 1000 / 120369
2018-Feb-01 18:43:38.258202 INFO    Imported block with index 2000 / 120369
2018-Feb-01 18:43:38.928033 INFO    Imported block with index 3000 / 120369
2018-Feb-01 18:43:39.454094 INFO    Imported block with index 4000 / 120369
2018-Feb-01 18:43:40.142969 INFO    Imported block with index 5000 / 120369
2018-Feb-01 18:43:40.830674 INFO    Imported block with index 6000 / 120369

After it completes it will start syncing incrementally like so:

2018-Feb-01 18:47:48.075930 INFO    Imported block with index 120000 / 120369
2018-Feb-01 18:47:48.860470 INFO    Core initialized OK
2018-Feb-01 18:47:48.860470 INFO    Initializing p2p server...

If you are using the GUI wallet, you can instead view the progress by opening log files and scrolling to the bottom.
